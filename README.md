# kalimero-pizza
A satirical website meant for a hypothetical pizza service called "Kalimero Pizzeria" based in Kosovska Mitrovica, featuring fake reviews and an option to sign up for the service. Upon sign up, users will be able to order pizza directly to their home, no matter their location in the world.
